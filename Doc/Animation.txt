# Описание формата файла анимации Simple Game Engine 1.17
#
#  Общая форма записи:
#  [Sprite Name] [Column X] [Column Y] [Visible Time]
#
#  Примеры:
#  Spr.Dino 0 1 100
#    Spr.Dino - Имя спрайта в таблице ресурсов
#    0 - Номер плитки по X  
#    1 - Номер плитки по Y
#    100 - Время видимости в милисекундах
#
#  Описание:
#  Пустые строки и начинающиеся с символа "#" пропускаются.
#
