#  Описание формата файла анимации Simple Game Engine 1.19
#
#  Общая форма записи:
#    [Sprite Name] [Column X] [Column Y] [Visible Time]
#      Sprite Name - Имя спрайта в таблице ресурсов
#      Column X - Номер плитки по X
#      Column Y - Номер плитки по Y
#      Visible Time - Время видимости в милисекундах
#
#  Примеры:
#    Spr.Dino 0 1 100
#
#  Дополнительно:
#    Пустые строки и начинающиеся с символа "#" пропускаются.
#
