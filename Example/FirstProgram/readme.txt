Существует два способа подключения движка к проекту:

  1. В качестве пакета SGE. Для этого нужно в Lazarus один раз открыть пакет SGE.
     Затем в Инспекторе проекта добавить в зависиимости этот пакет.

  2. Добавить в параметры проекта пути к модулям SGE следующих каталогов: 
     Source\, 
     Source\SimpleTools,
     Source\ThirdParty\dglOpenGL,
     Source\ThirdParty\gdiPlus,
     Source\ThirdParty\OpenAL.


В движке есть несколько базовых классов:
  1. Window - Окно для вывода
  2. Graphic - Методы отрисовки
  3. Sound - Проигрывание звука
  4. Resources - Менеджер ресурсов
  5. Joysticks - Доступ к джойстикам


После создания дочернего класса, нужно инициализировать Окно и графику.
По умолчанию окно скрыто и нужно его показать Window.Show. Затем следует
запуск бесконечного цикла выборки сообщений и отрисовка кадра. Удобнее всего
перекрыть метод Create и добавить код тонкой настройки поведения. Большинство
возможностей движка состоит из виртуальных методов которые необходимо 
переопределять в своём классе. В данном примере показан способ отрисовки сцены.
В проекте отключена отладочная информация в опциях проекта.
