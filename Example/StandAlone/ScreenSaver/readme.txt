Как создать шаблон для проекта
1. Создаём пустой проект в Lazarus
2. Подключаем пакет SGE
3. Создаём класс и наследуем от TsgeScreenSaver
4. Добавляем директиву компилятора {$R SaverName.rc}
5. В параметрах проекта в Имени исполняемого файла добавляем в
   конец .scr (Расширение для хранителей)
5. Переопределяем метод Draw и Tick
6. Перед запуском хранителя экрана нужно настроить 
   интервал вызова метода Tick через свойство TickDelay.
7. Далее создаём объект
8. Запускаем хранитель методом Run.
9. Строка в файле ресурсов SaverName.rc будет показываться
   в диалоге настроки хранителя в Операционной сиситеме.
10.После создания exe переименуте его в *.Scr - это и будет
   готовый хранитель.



Каждый хранитель запускается с параметрами командной строки:
  /S - Обычный запуск
  /С - Диалог настройки хранителя
  /P - Режим предпросмотра в диалоге настройки ОС

В класс TsgeScreenSaver встроен весь необходимый код для
взаимодействия с операционной системой. Остаётся только
реальзовать метод отрисовки и обсчёта тиков. В примере используется
массив квадратов, постепенно заполняющих экран.

Предусмотрено 2 виртуальных функции:
  LoadSaverSettings - тут происходит загрузка параметров
                      хранителя из файла или реестра.
StartSaverConfigBox - Вызов настройки хранителя из диалога
                      в ОС.

ВАЖНО!
  Замечен баг. В проекте используется файл ресурсов *.RC. Встроенный
  компилятор WindRes наотрез отказывается компилировать, если в путях
  к файлам присутствуют Русские буквы.
